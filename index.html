Angular Checkbox Range

AngularJS module for add the ability to check a range of checkboxes

Download this project as a .zip file Download this project as a tar.gz file

Usage

The direcitves multi-check-group and multi-check define which checkboxes should have the ability.

<div multi-check-group>
          <div ng-repeat="p in people">
            <label>
              <input type="checkbox" multi-check ng-model="p.checked" /> {{p.name}}
            </label>
          </div>
        </div>

Follow the instructions below and observe all checkboxes in between become checked.

Unchecking works too!

Follow the instructions below and observe all checkboxes in between become unchecked.

You can define multiple groups!

<div ng-controller="MyController">
          <div multi-check-group>
            <div ng-repeat="p in people">
              <label>
                <input type="checkbox" multi-check ng-model="p.checked" /> {{p.name}}
              </label>
            </div>
          </div>
          <div multi-check-group>
            <div ng-repeat="p in people2">
              <label>
                <input type="checkbox" multi-check ng-model="p.checked" /> {{p.name}}
              </label>
            </div>
          </div>
        </div>

Follow the instructions below and observe the checkboxes inbetween are not checked because they are in different groups.
